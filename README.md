
# Verilog Watchdog Timer

## Overview
This project implements a Watchdog Timer using Verilog HDL. A watchdog timer is used to monitor system operation and generate a reset signal when the system fails to respond within a specified time period.

## Project Objectives
- Design a Watchdog Timer using Verilog HDL
- Monitor system activity
- Detect timeout conditions
- Generate reset signals when required

## Files Included

| File Name | Description |
|------------|-------------|
| watchdog.v | Main RTL design of Watchdog Timer |
| watchdog_tb.v | Testbench for simulation and verification |

## Features
- Timeout detection
- Automatic reset generation
- RTL implementation in Verilog
- Simulation-based verification

## Tools Used
- Verilog HDL
- Xilinx Vivado

## Applications
- Embedded Systems
- Microcontroller-based Systems
- FPGA Designs
- Digital VLSI Systems
